arrPartitionIndex

Linear-time Partitioning. Runtime: O(n). Chooses a pivot and re-arrages the array so that everything on the left is <= pivot and everything on the right is > pivot

arrPartitionIndex(arr: Array, lo: number, hi: number): number
Parameters
arr (Array)
lo (number) start index
hi (number) end index
Returns
number: pivot index
